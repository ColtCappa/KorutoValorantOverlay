# KorutoValorantOverlay
A short custom VALORANT rank overlay for streaming


This tool was created with much love by the community for the community, utilizing [Henrik's Valorant API](https://github.com/Henrik-3/unofficial-valorant-api).
We hope you appreciate it!


Setup:
Add *overlay.html* file as a browser source into OBS and check the local file box.
Set you API key from [Henrik's Valorant API](https://github.com/Henrik-3/unofficial-valorant-api) in *KorutoMMR.py*.
Add your region, riot ID and tag in *KorutoMMR.py*.

Now your rank should appear in the html on your obs page and should auto-update!

Run this order:

Start Python script
Start OBS
Start stream
